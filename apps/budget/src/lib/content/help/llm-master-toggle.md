# LLM Features

Master toggle to enable or disable all language model features.

## When Enabled

- AI providers can enhance ML features
- Richer, more contextual suggestions
- Natural language transaction parsing
- Intelligent anomaly explanations

## When Disabled

- Only local ML features are used
- No external API calls
- Full privacy - data stays local
- Lower operating costs

## Requirements

To use LLM features, you need:
1. This toggle enabled
2. At least one provider configured
3. A default provider selected

## Tips

- Enable for the most intelligent experience
- Disable if you prefer local-only processing
- Individual features can be configured below
