# Transaction Toolbar

The toolbar provides powerful controls for filtering, searching, and customizing your transaction view.

## Toolbar Sections

### Saved Views
Pre-configured filters and column layouts:
- **Account** - Default view for this account
- **Custom Views** - Your saved configurations
- **Edit Views** - Modify saved views

### Filters
Narrow down transactions by:
- Date range
- Amount range
- Category
- Payee
- Status

### View Options
Customize the display:
- Show/hide columns
- Change sort order
- Adjust column widths

## Creating Views

1. Apply filters and column settings
2. Click **New View**
3. Name your view
4. Save for quick access

## Keyboard Shortcuts

- [[Ctrl+F]] - Focus search/filter
- [[Ctrl+S]] - Save current view
- [[Escape]] - Clear filters

## Tips

- Combine multiple filters for precision
- Save frequently used filter combinations
- Views are account-specific
- Reset to default with the Reset button
