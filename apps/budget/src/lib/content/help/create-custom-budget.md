# Create Custom Budget

Navigate to the budget creation page to set up a new budget with complete control over all settings.

## When to Use

Choose this option when:
- No template matches your needs
- You have specific requirements
- You want to set up an unusual budget type
- You prefer to configure everything yourself

## What You Can Do

On the budget creation page, you can:
- Set a custom budget name and description
- Choose your budget type and category
- Set allocation amounts and periods
- Configure enforcement and rollover settings
- Link accounts and categories
