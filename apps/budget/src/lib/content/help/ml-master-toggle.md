# Machine Learning Features

Master toggle to enable or disable all ML-powered features.

## When Enabled

- Forecasting predicts future spending
- Anomaly detection flags unusual transactions
- Similarity matching groups related payees
- User behavior learning improves suggestions

## When Disabled

- The app runs in manual mode only
- No automatic predictions or suggestions
- Lower resource usage
- Full privacy - no ML processing

## Tips

- Keep enabled for the best experience
- Disable if you prefer manual control
- Individual features can be toggled separately below
