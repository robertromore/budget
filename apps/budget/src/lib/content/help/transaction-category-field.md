# Category Field

Categories help organize transactions for budgeting and reporting.

## Selection Methods

1. **Type to search** - Start typing to filter categories
2. **Browse list** - Scroll through all available categories
3. **Auto-assign** - May be filled based on payee defaults

## Category Hierarchy

Categories can be organized in groups:
- **Parent categories** (e.g., "Food")
- **Sub-categories** (e.g., "Food > Groceries", "Food > Dining Out")

## Smart Categorization

When AI intelligence is enabled:
- Categories are suggested based on payee
- Similar transactions inform recommendations
- Confidence scores indicate suggestion quality

## Tips

- Leave blank for uncategorized transactions
- Create categories that match your budget structure
- Consistent categorization improves analytics
- You can bulk-recategorize later from the Transactions tab

## Budget Integration

- Categories link to budgets for tracking
- Spending against budget limits is calculated automatically
- Over-budget categories are highlighted in reports

## Keyboard Shortcuts

- [[Down Arrow]] - Open category dropdown
- [[Enter]] - Select highlighted category
- [[Escape]] - Close dropdown
