# Display Settings

Configure how dates, numbers, and currencies are displayed throughout the app.

## Date Format

Choose how dates are shown:
- **MM/DD/YYYY** - US format (12/25/2024)
- **DD/MM/YYYY** - European format (25/12/2024)
- **YYYY-MM-DD** - ISO format (2024-12-25)

## Currency Symbol

Set your preferred currency symbol (e.g., $, €, £). This appears before amounts throughout the app.

## Number Format

Choose your locale-based number formatting:
- **en-US** - US format (1,234.56)
- **de-DE** - German format (1.234,56)
- **fr-FR** - French format (1 234,56)

## Decimal Places

Toggle whether to show cents/decimal places in currency amounts. When disabled, amounts are rounded to whole numbers.

## Header Layout

### Header Actions Mode

Control where action buttons appear:
- **Keep on page** - Buttons stay in the page content
- **Secondary to header** - Move secondary actions to header
- **All to header** - Move all actions to header bar

### Header Actions Display

Choose how header buttons appear:
- **Icon with text** - Show both icon and label
- **Icon only** - Compact icons without labels

### Header Tabs Mode

Control where page tabs appear:
- **Keep on page** - Tabs remain in page content
- **Move to header** - Tabs appear in the header bar

## Table Options

### Table Display Options

Choose how table options are presented:
- **Popup dropdown** - Options appear in a dropdown menu
- **Responsive sheet** - Options open in a slide-out sheet

## Tips

- Date format should match your regional preference
- Number format affects how amounts are parsed during import
- Moving items to the header saves vertical space
