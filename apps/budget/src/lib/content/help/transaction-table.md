# Transaction Table

The transaction table displays all transactions for the selected account with powerful filtering and sorting capabilities.

## Table Features

- **Sorting** - Click column headers to sort by that field
- **Filtering** - Use the toolbar to filter by date, amount, or category
- **Selection** - Check boxes to select multiple transactions for bulk actions

## Column Information

- **Date** - When the transaction occurred
- **Description** - Transaction details from your bank
- **Payee** - The merchant or vendor
- **Category** - Budget category assignment
- **Amount** - Transaction value

## Keyboard Shortcuts

- [[Ctrl+F]] - Focus the search/filter field
- [[Space]] - Toggle selection on focused row

## Tips

- Use date range filters to focus on specific periods
- Bulk edit categories for similar transactions
- Export filtered data for external analysis
