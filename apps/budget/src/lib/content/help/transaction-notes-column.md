# Notes Column

The notes column displays optional memos or descriptions for transactions.

## Display

- Shows note text (truncated if long)
- Hover to see full note in tooltip
- Empty cells show placeholder or dash
- Scheduled transactions show sticky note icon

## Inline Editing

Click any notes cell to edit:
1. A text input appears
2. Type your note
3. Press [[Enter]] or click outside to save
4. Press [[Escape]] to cancel

## Common Uses

- **Expense details** - "Lunch with client - Johnson account"
- **Receipt references** - "Receipt #12345"
- **Context notes** - "Emergency repair - covered by warranty"
- **Split details** - "My portion of group dinner"

## Search Integration

Notes are included in transaction search:
- Search keywords match note content
- Use search to find specific memos
- Combine with filters for precision

## Features

- Multi-line text support
- No character limit
- Searchable content
- Visible in transaction details

## Sorting

- Notes column sorting is disabled
- Use search to find specific notes

## Tips

- Notes are optional - skip if not needed
- Keep notes concise for easier scanning
- Use consistent formats for similar transactions
- Notes appear in exports and reports
