# Payee Column

The payee column identifies the merchant, vendor, or person for each transaction.

## Display

- Shows the payee name
- Scheduled transactions display with a hand/coins icon
- Transfers show linked account indicator

## Inline Editing

Click any payee cell to edit:
1. A searchable dropdown appears
2. Type to filter existing payees
3. Select from suggestions or create new
4. Changes save automatically

## Smart Features

When AI intelligence is enabled:
- **Auto-complete** - Suggests matches as you type
- **Fuzzy matching** - Finds payees even with typos
- **Similar payees** - Groups variations (e.g., "AMZN" and "Amazon")

## Filtering

Filter by payee:
- **Is** - Show only selected payees
- **Is not** - Exclude selected payees
- Multi-select supported

## Sorting

- Alphabetical by payee name
- Click header to toggle ascending/descending
- Null payees sort to end

## Tips

- Selecting a payee can auto-fill the category based on defaults
- Consistent naming improves reports and analytics
- Use the Payees page to merge duplicates
- Payee names are shared across all accounts
