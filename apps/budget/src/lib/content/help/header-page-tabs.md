# Page Tabs

Page tabs provide context-specific navigation within a section of the app.

## How They Work

- Tabs appear for pages with multiple views
- Click a tab to switch views
- Current tab is highlighted

## Common Tab Locations

### Account Detail Page

- Transactions
- Analytics
- Schedules
- Budgets
- Import
- Settings

### Budget Detail Page

- Overview
- Transactions
- Analytics
- Settings

### Payee Detail Page

- Transactions
- Intelligence
- Settings

## Display Options

Configure in Settings > Display:

- **Show in header** - Tabs appear in header bar
- **Show on page** - Tabs appear within page content
- **Off** - Hide header tabs

## Keyboard Navigation

- [[Left Arrow]] / [[Right Arrow]] - Switch tabs (when focused)
- Numbers [[1-9]] - Jump to specific tab

## Tips

- Tabs remember your last selection per page
- Some tabs show badge counts
- Look for tabs when exploring new pages
- Customize display in Settings > Display
