# Intelligence Input Mode

Intelligence mode provides AI-powered suggestions as you fill out forms.

## Toggling the Mode

- Click the **brain** icon in the header
- Or press [[Ctrl+Shift+I]] / [[Cmd+Shift+I]]
- Green dot indicates active mode

## How It Works

When active, form fields show:

1. **Smart suggestions** as you type
2. **Auto-complete** based on history
3. **Category recommendations** for transactions
4. **Payee normalization** suggestions

## Supported Fields

Intelligence mode enhances:

- **Payee input** - Merchant name suggestions
- **Category selector** - Smart categorization
- **Amount field** - Pattern detection
- **Notes** - Context-aware suggestions

## Suggestion Types

| Type | Description |
|------|-------------|
| **History** | Based on your past entries |
| **Pattern** | Detected spending patterns |
| **AI** | Machine learning suggestions |
| **Similar** | Matching existing entries |

## Accepting Suggestions

- Click a suggestion to accept
- Press [[Tab]] to accept top suggestion
- Keep typing to ignore

## Performance

- Suggestions appear in real-time
- Works offline with cached patterns
- No data sent to external services

## Keyboard Shortcut

- [[Ctrl+Shift+I]] / [[Cmd+Shift+I]] - Toggle mode

## Tips

- Enable for faster data entry
- Disable if suggestions are distracting
- Configure in Settings > ML Settings
- Works best with more transaction history
