# Amount Field

Enter the monetary value of the transaction.

## Sign Convention

- **Negative values** (-$50.00) = Expenses, payments, withdrawals
- **Positive values** ($50.00) = Income, deposits, refunds

## Input Methods

1. **Direct entry** - Type the number (e.g., "50" or "50.00")
2. **Increment buttons** - Use +/- buttons for quick adjustments
3. **Calculator mode** - Type expressions like "50+25"

## Formatting

- Currency symbol is added automatically
- Decimal places follow your display settings
- Thousands separators are applied automatically

## Tips

- You can paste amounts from other applications
- The field strips non-numeric characters automatically
- Use [[Tab]] to move to the next field
- Leave as 0 if you're unsure - you can edit later

## Keyboard Shortcuts

- [[Up Arrow]] - Increase by $1
- [[Down Arrow]] - Decrease by $1
- [[Shift+Up]] - Increase by $10
- [[Shift+Down]] - Decrease by $10
