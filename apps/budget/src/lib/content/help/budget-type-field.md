# Budget Type

Select how you want to track and enforce this budget.

## Available Types

### Account Monthly
Track total spending from a specific account each month. Best for:
- Monitoring overall account activity
- Controlling credit card spending
- Tracking checking account outflows

### Category Monthly
Track spending in a specific category each month. Best for:
- Expense categories like groceries, entertainment
- Recurring bills tracking
- Discretionary spending limits

### Goal-Based
Track progress toward a savings target with a deadline. Best for:
- Emergency fund savings
- Large purchases (car, vacation)
- Debt payoff targets
- Financial milestones

### Envelope
Allocate a fixed amount that doesn't reset - like physical envelope budgeting. Best for:
- Irregular expenses (car maintenance)
- Annual subscriptions
- Variable categories

## Enforcement

Each type can have different enforcement levels:
- **Strict** - Hard limit, warns before exceeding
- **Flexible** - Soft guidance, shows warnings
- **Tracking** - No limits, just monitoring
