# Add Transaction Dialog

This dialog provides multiple ways to enter new transactions into your account.

## Entry Tabs

### Manual Entry
Traditional form with full control over:
- Amount, date, payee, category
- Budget allocation
- Notes and memo

### Transfer
Simplified form for moving money between your accounts:
- Select source and destination accounts
- Enter transfer amount
- Creates matching transactions in both accounts

### Guided Entry
Step-by-step wizard that:
- Asks questions one at a time
- Provides smart suggestions
- Validates each step before proceeding

## Form Fields

| Field | Required | Description |
|-------|----------|-------------|
| Amount | Yes | Transaction value (negative = expense) |
| Date | Yes | When the transaction occurred |
| Payee | No | Merchant or vendor name |
| Category | No | Budget category for tracking |
| Budget | No | Optional budget allocation |
| Notes | No | Additional description or memo |

## Tips

- Press [[Enter]] to submit the form
- Press [[Escape]] to cancel and close
- Payee suggestions appear as you type
- Recent payees are shown first
