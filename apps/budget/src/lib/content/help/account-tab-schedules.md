# Schedules Tab

Create and manage scheduled transactions for this account.

## What Are Schedules?

Schedules automate recurring transactions:
- Monthly bills (rent, utilities)
- Subscriptions (streaming, software)
- Regular transfers (savings contributions)
- Periodic expenses (quarterly insurance)

## Schedule List

View all schedules linked to this account:
- **Name** - Description of the schedule
- **Amount** - Transaction value
- **Frequency** - How often it repeats
- **Next Date** - When it will next occur
- **Status** - Active, paused, or completed

## Creating Schedules

1. Click "Add Schedule"
2. Set payee and amount
3. Choose frequency (daily, weekly, monthly, yearly)
4. Set start date and optional end date
5. Configure auto-posting preferences

## Schedule Actions

- **Edit** - Modify schedule details
- **Pause** - Temporarily stop scheduling
- **Delete** - Remove the schedule
- **Post Now** - Create transaction immediately

## Recommendations

The system can suggest schedules based on:
- Detected recurring patterns
- Similar transaction frequencies
- Payment history analysis

## Tips

- Review upcoming schedules regularly
- Pause schedules during billing changes
- Use for predictable expenses only
- Set reminders before large scheduled payments
