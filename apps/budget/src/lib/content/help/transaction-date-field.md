# Date Field

Select the date when the transaction occurred or is scheduled to occur.

## Date Selection

- **Calendar picker** - Click to open a visual calendar
- **Direct entry** - Type the date in your preferred format
- **Quick shortcuts** - Use keyboard shortcuts for common dates

## Keyboard Shortcuts

- [[T]] - Set to today
- [[Y]] - Set to yesterday
- [[Left Arrow]] - Previous day
- [[Right Arrow]] - Next day
- [[Up Arrow]] - Previous week
- [[Down Arrow]] - Next week

## Date Formats

The date displays according to your settings in:
Settings > Display > Date Format

Common formats:
- MM/DD/YYYY (US)
- DD/MM/YYYY (UK/EU)
- YYYY-MM-DD (ISO)

## Tips

- Defaults to today's date
- Future dates create scheduled transactions
- Past dates are allowed for backdating
- The date affects which budget period the transaction counts toward
