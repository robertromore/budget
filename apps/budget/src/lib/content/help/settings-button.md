# Settings Button

Click to open the application settings page.

## Settings Sections

### Appearance

- Theme mode (light/dark/system)
- Color scheme
- Font preferences

### Display

- Date format
- Currency symbol
- Number formatting
- Decimal places
- Header options

### ML Settings

- Intelligence features
- Anomaly detection
- Forecast settings
- Similarity thresholds

### LLM Settings

- AI provider configuration
- API keys
- Feature modes

### Import Profiles

- CSV import mappings
- Column configurations
- Default settings

### Advanced

- Data management
- Export options
- Danger zone

## Quick Access

- Click the **gear** icon in the header
- Direct link: `/settings`

## Tips

- Settings are workspace-specific
- Changes take effect immediately
- Use keyboard [[Ctrl+,]] on some pages
