# Schedules Table

The schedules table displays all your recurring transactions with their status and upcoming dates.

## Table Columns

- **Name** - Schedule name
- **Amount** - Recurring amount
- **Frequency** - How often it repeats
- **Next Date** - When the next occurrence is due
- **Status** - Active, paused, or completed
- **Actions** - Quick action buttons

## Schedule Status

- **Active** - Currently running, will create transactions
- **Paused** - Temporarily stopped
- **Completed** - Finished all occurrences

## Available Actions

- **View** - See full schedule details
- **Edit** - Modify schedule settings
- **Skip** - Skip the next occurrence
- **Delete** - Remove the schedule

## Tips

- Sort by next date to see upcoming payments
- Pause schedules for temporary subscriptions
- Use bulk delete to clean up old schedules
