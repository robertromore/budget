# Payees Table

The payees table displays all your payees with key information and quick actions.

## Table Columns

- **Name** - Payee name (click to view details)
- **Category** - Default category for auto-categorization
- **Transactions** - Number of transactions with this payee
- **Average** - Average transaction amount
- **Last Transaction** - Most recent transaction date
- **Actions** - Quick action buttons

## Available Actions

- **View** - See full payee details and transaction history
- **Edit** - Modify payee information
- **Analytics** - View spending charts and patterns for this payee
- **Delete** - Remove the payee (with confirmation)

## Bulk Actions

Select multiple payees using checkboxes to:
- Delete multiple payees at once
- Export selected payees

## Search & Filter

Use the search bar to find payees by:
- Name
- Category
- Notes

## Tips

- Sort by transaction count to find your most frequent payees
- Click column headers to sort the table
- Use bulk selection for cleanup operations
