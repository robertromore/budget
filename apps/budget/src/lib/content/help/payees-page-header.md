# Payees Page

Payees are merchants, companies, or people you transact with. This page helps you manage and organize all your payees.

## Overview

Use this page to:
- View all your payees in one place
- Add new payees manually
- Set default categories for automatic categorization
- View transaction statistics for each payee

## Available Actions

- **Manage Categories** - Organize payees into categories
- **Analytics** - View spending insights across all payees
- **Add Payee** - Create a new payee

## Payee Features

- **Default Category** - Automatically categorize transactions from this payee
- **Default Account** - Associate payee with a specific account
- **Notes** - Add details about the payee
- **Contact Info** - Store address, phone, email

## Tips

- Set default categories for frequently used payees to save time
- Use the analytics feature to identify top spending patterns
- Merge duplicate payees to keep your data clean
