# Import Tab

Bring in transactions from your bank or financial institution.

## Supported Formats

- **CSV** - Comma-separated values (most common)
- **OFX** - Open Financial Exchange
- **QFX** - Quicken Financial Exchange
- **QIF** - Quicken Interchange Format

## Import Process

1. **Upload File** - Drag & drop or click to select
2. **Match Columns** - Map file fields to transaction fields
3. **Preview** - Review detected transactions
4. **Import** - Add transactions to your account

## Import Profiles

Save column mappings for repeated imports:
- Create profiles for each bank format
- Auto-detect files by name pattern
- Reuse settings for future imports

## Duplicate Detection

The system checks for:
- Same date, amount, and payee
- Similar transactions within range
- Already imported entries

## Smart Features

- **Auto-categorization** - Suggest categories for new payees
- **Payee Matching** - Link to existing payees
- **Pattern Learning** - Improve suggestions over time

## Tips

- Export from bank in CSV format when possible
- Check date format matches your region
- Review duplicates before confirming
- Create profiles for regular bank downloads
- Enable ML features for smarter matching
