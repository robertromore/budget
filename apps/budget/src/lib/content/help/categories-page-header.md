# Categories Page

Categories help you organize and classify your transactions for better spending analysis.

## Overview

Categories allow you to:
- Group similar transactions together
- Track spending by category
- Create budgets based on categories
- Generate spending reports and charts

## Available Actions

- **Seed Default Categories** - Create common category presets
- **Group Management** - Organize categories into groups
- **Analytics** - View spending breakdown by category
- **Add Category** - Create a new category

## Category Features

- **Groups** - Organize categories into logical groups
- **Icons** - Visual identifiers for quick recognition
- **Colors** - Color-code categories for charts
- **Parent Categories** - Create hierarchical structures

## Tips

- Use groups to organize related categories (e.g., "Housing", "Transportation")
- Set up categories before importing transactions for auto-categorization
- Review the hierarchy view to understand category relationships
