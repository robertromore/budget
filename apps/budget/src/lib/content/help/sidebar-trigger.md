# Toggle Sidebar

Click to show or hide the navigation sidebar.

## Sidebar States

- **Expanded** - Full sidebar with labels visible
- **Collapsed** - Icons only, more screen space
- **Hidden** - Sidebar completely hidden (mobile)

## When to Use

- Hide sidebar for more workspace on smaller screens
- Collapse for quick access with minimal space
- Expand when navigating between sections

## Keyboard Shortcut

- [[Ctrl+B]] / [[Cmd+B]] - Toggle sidebar visibility

## Mobile Behavior

On mobile devices:

- Sidebar opens as an overlay
- Tap outside to close
- Swipe to dismiss

## Tips

- Sidebar state is remembered across sessions
- Hover on collapsed sidebar to see labels
- Use keyboard shortcut for quick toggle
