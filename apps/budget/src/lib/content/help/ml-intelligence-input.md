# Intelligence Input Mode

An overlay mode that highlights form fields with AI/ML capabilities.

## How to Activate

Press **Cmd/Ctrl + Shift + I** to toggle the mode on or off.

## What It Does

When active:
- Form fields with AI capabilities are highlighted
- Click a field to get intelligent suggestions
- Choose between ML and LLM for each field

## Visual Indicators

- **Highlighted fields**: Have AI capabilities
- **ML icon**: Uses local machine learning
- **LLM icon**: Uses language models

## Keyboard Shortcuts

While a field is highlighted:
- **M**: Switch to ML mode
- **L**: Switch to LLM mode
- **Esc**: Exit intelligence input mode

## Tips

- Great for quickly filling forms with suggestions
- Combine with tab navigation for efficient data entry
- The mode remembers your preferences per field
