# Search Provider

Choose which service to use for web searches.

## Available Providers

### DuckDuckGo
- **Free**: No API key required
- **Privacy-focused**: Doesn't track searches
- Uses HTML scraping, may be slower

### Brave Search API
- **2,000 free queries/month**
- Requires API key from brave.com/search/api
- Fast and reliable API

### Ollama Web Search
- Cloud-based search service
- Requires Ollama API key
- Good for users already using Ollama

## Getting API Keys

### Brave Search
1. Visit brave.com/search/api
2. Create a free account
3. Generate an API key
4. Enter it in the settings

## Tips

- Start with DuckDuckGo if you don't want to set up an API key
- Brave Search is more reliable for heavy usage
- API keys are stored encrypted
