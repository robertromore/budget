# Account Tabs

Tabs provide different views and actions for your account.

## Available Tabs

- **Transactions** - View and manage all account transactions
- **Import** - Import transactions from bank files
- **Settings** - Configure account preferences
- **Intelligence** - AI-powered insights and predictions

## Tips

- The current tab is reflected in the URL for easy bookmarking
- Switch tabs using keyboard shortcuts or clicking the tab headers
