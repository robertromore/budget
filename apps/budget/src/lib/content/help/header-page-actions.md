# Page Actions

Page actions are context-specific buttons that appear in the header based on the current page.

## How They Work

- Actions change depending on which page you're viewing
- Primary actions appear prominently
- Secondary actions may be in a dropdown

## Common Actions by Page

### Accounts Page

- Add Account

### Account Detail

- Add Transaction
- Import Transactions

### Budgets Page

- Create Budget
- Use Template

### Categories Page

- Add Category
- Manage Groups
- Add Default Categories

### Payees Page

- Add Payee
- Merge Payees
- AI Enhancement

### Schedules Page

- Add Schedule

## Display Modes

Configure in Settings > Display:

- **All actions** - Show all page actions
- **Secondary only** - Only non-primary actions
- **Off** - Hide page actions from header

## Icon vs Text

Actions can display as:

- **Icon only** - Compact, hover for label
- **Icon + Text** - Full labels visible

## Tips

- Actions are page-specific
- Look here for the main action for any page
- Keyboard shortcuts may be available
- Customize display in Settings > Display
