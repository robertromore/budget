# View Options

Customize which columns are visible and how they're displayed.

## Column Visibility

Show or hide columns:
- Date
- Description
- Payee
- Category
- Amount
- Balance
- Status
- Notes
- Actions

## Column Order

Drag columns to reorder:
1. Click and hold column header
2. Drag to new position
3. Release to drop

## Column Width

Resize columns:
1. Hover over column border
2. Drag to adjust width
3. Double-click to auto-fit

## Density Options

- **Compact** - More rows visible
- **Normal** - Balanced spacing
- **Comfortable** - Easier reading

## Saving Layout

Your column configuration is automatically saved:
- Per-account settings
- Named views remember layout
- Export includes visible columns only

## Tips

- Hide columns you rarely use
- Pin important columns to left
- Reset to defaults if needed
- Different views for different tasks
