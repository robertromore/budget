# Web Search Enrichment

Automatically search the web to find contact information for payees.

## What It Does

When enabled, the app can search for:
- Business phone numbers
- Physical addresses
- Website URLs
- Business hours

## Privacy Note

- Searches are performed on-demand, not automatically
- Only payee names are searched
- No personal financial data is shared

## When to Enable

- You want to quickly find contact info for merchants
- You need to dispute charges or contact businesses
- You want richer payee profiles

## When to Disable

- You prefer to add contact info manually
- You have limited internet connectivity
- You want maximum privacy
