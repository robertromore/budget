# Payees List

Payees are the merchants, vendors, and individuals you transact with.

## Payee Features

- **Auto-categorization** - Transactions automatically categorize based on payee
- **Spending insights** - See how much you spend with each payee
- **Smart suggestions** - AI-powered payee matching for new transactions

## Managing Payees

- Merge duplicate payees to consolidate data
- Set default categories for automatic assignment
- View transaction history by payee

## Tips

- Clean up imported payee names for better readability
- Use payee insights to identify subscription costs
- The system learns your categorization preferences over time
