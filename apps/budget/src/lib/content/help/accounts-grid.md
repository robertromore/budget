# Accounts Grid

The accounts grid displays all your financial accounts in an organized card layout.

## Card Information

Each account card shows:
- **Account name** - Click to view transactions
- **Account icon** - Visual identifier with custom color
- **Account type** - Checking, savings, credit card, etc.
- **Institution** - Bank or financial institution name
- **Balance** - Current account balance (green for positive, red for negative)
- **Notes** - Brief description or notes about the account

## Available Actions

- **Click account name** - Navigate to account transactions
- **Edit button** - Modify account settings
- **Delete button** - Remove the account (with confirmation)

## Visual Indicators

- **Color-coded border** - Custom color on the left edge for quick identification
- **Balance colors** - Green for positive, red for negative balances
- **Closed badge** - Indicates accounts that have been closed

## Tips

- The grid automatically adjusts to screen size (1-4 columns)
- Hover over truncated notes to see the full text
- Account numbers show only the last 4 digits for security
