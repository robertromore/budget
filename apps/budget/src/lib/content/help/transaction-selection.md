# Row Selection

Select one or more transactions to perform bulk operations.

## Selection Methods

### Single Selection
- Click the checkbox on any row
- Click toggles selection on/off

### Select All on Page
- Click the header checkbox
- Selects all visible transactions on current page
- Click again to deselect all

### Range Selection
- Click first row checkbox
- Hold [[Shift]] and click last row
- All rows between are selected

## Selection Indicator

- Header checkbox shows current state:
  - Empty: No rows selected
  - Check: All page rows selected
  - Dash (indeterminate): Some rows selected

## Selection Limits

- Scheduled transactions cannot be selected
- Selection persists during filtering
- Pagination resets visible selection

## Bulk Actions Available

When rows are selected:
- **Categorize** - Assign category to all
- **Delete** - Remove selected transactions
- **Export** - Export selected to file
- **Status change** - Update status for all

## Keyboard Shortcuts

- [[Space]] - Toggle selection on focused row
- [[Ctrl+A]] - Select all on page
- [[Escape]] - Clear selection

## Tips

- Selection count appears in toolbar
- Filter first, then select for precision
- Bulk categorization saves time
- Review selections before destructive actions
