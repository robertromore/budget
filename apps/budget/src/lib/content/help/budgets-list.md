# Budgets List

Budgets help you plan and control spending by setting limits for different expense categories.

## Budget Status

- **On Track** - Spending is within your planned amount
- **At Risk** - Approaching the budget limit
- **Over Budget** - Spending exceeds the planned amount

## Creating Budgets

1. Click "Add Budget" to create a new category
2. Set a monthly spending limit
3. Assign transactions to track spending

## Tips

- Review budgets regularly to adjust for changing needs
- Use the intelligence features to predict budget overruns
- Rollover unused amounts to the next month if desired
