# Intelligence Tab

Machine learning-powered analysis of your account activity.

## Insights Available

### Spending Patterns
- Recurring transaction detection
- Unusual activity alerts
- Spending trend analysis

### Anomaly Detection
- Suspicious transactions flagged
- Duplicate charge warnings
- Amount variance alerts

### Smart Suggestions
- Category recommendations
- Budget optimization tips
- Savings opportunities

### Forecasting
- Predicted future balance
- Expected recurring charges
- Cash flow projections

## How It Works

The intelligence system analyzes:
- Transaction history patterns
- Payee behavior
- Seasonal variations
- Similar user trends

## Confidence Levels

Each insight shows confidence:
- **High** - Strong historical pattern
- **Medium** - Likely but verify
- **Low** - Speculative suggestion

## Tips

- Review anomalies regularly
- Accept/reject suggestions to improve accuracy
- Check forecasts before large purchases
- Enable ML features in Settings for full functionality
