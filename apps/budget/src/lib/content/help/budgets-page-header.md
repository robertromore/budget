# Budgets Page

The Budgets page helps you manage your spending by setting up budgets for different categories.

## Overview

Budgets allow you to:
- Set spending limits for categories
- Track actual spending vs. budgeted amounts
- Get alerts when approaching or exceeding limits
- Analyze spending patterns over time

## Available Actions

- **Templates** - Choose from pre-built budget templates
- **Create Budget** - Set up a new budget from scratch

## Budget Types

- **Category budgets** - Track spending for a specific category
- **Account budgets** - Monitor spending from a specific account
- **Envelope budgets** - Allocate funds to virtual envelopes

## Tips

- Start with a few core budgets (groceries, dining, entertainment)
- Review and adjust budgets monthly based on actual spending
- Use the recommendations tab for AI-powered budget suggestions
