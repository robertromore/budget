# Payee Intelligence Summary

This card uses machine learning to provide intelligent insights about your payees and spending patterns.

## Key Metrics

- **Total Payees** - Complete count of all payees, with active count highlighted
- **With Defaults** - Payees that have default categories or accounts configured for faster transaction entry
- **Avg Transactions** - Average number of transactions per payee, indicating how frequently you use each vendor
- **Need Attention** - Payees that might benefit from review or configuration updates

## Top Categories

Shows the most common categories your payees fall into, helping you understand your spending distribution.

## Actions

- **Full Analytics Dashboard** - Access comprehensive payee analytics with charts and detailed breakdowns
- **Manage Payees** - Navigate to the payees list for full management capabilities

## Tips

- Configure defaults for frequently used payees to speed up transaction entry
- Review payees needing attention to ensure accurate categorization
- Use the analytics dashboard for deeper insights into spending trends
