# Help Mode

Help mode provides contextual documentation for any element in the interface.

## How to Use

1. Click the help button to activate help mode
2. Move your cursor over any highlighted element
3. Click an element to view its documentation
4. Press [[Escape]] to exit help mode

## Keyboard Navigation

- [[Tab]] / [[Arrow Keys]] - Navigate between elements
- [[Enter]] - Open documentation for selected element
- [[Escape]] - Close documentation or exit help mode
- [[Ctrl+Shift+/]] - Toggle help mode from anywhere

## Tips

- Elements with documentation are highlighted with a subtle outline
- The currently selected element shows a brighter highlight
