# Date Format

Choose how dates are displayed throughout the app.

## Available Formats

- **MM/DD/YYYY** - US format (12/31/2024)
- **DD/MM/YYYY** - European format (31/12/2024)
- **YYYY-MM-DD** - ISO format (2024-12-31)

## Tips

- Choose the format you're most familiar with
- This affects all date displays in the app
- Does not affect how dates are stored, only how they're shown
