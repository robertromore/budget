# Category Column

The category column assigns transactions to budget categories for tracking and reporting.

## Display

- Shows category name with color indicator
- Sub-categories display with hierarchy (e.g., "Food > Groceries")
- Uncategorized transactions show placeholder text

## Inline Editing

Click any category cell to edit:
1. A categorized dropdown appears
2. Browse or search categories
3. Select to assign immediately
4. Click clear icon to remove category

## Category Features

- **Color coding** - Each category has a distinct color
- **Hierarchy** - Categories can have parent/child relationships
- **Smart suggestions** - AI can suggest categories based on payee

## Filtering

Filter by category:
- **Is** - Show only selected categories
- **Is not** - Exclude selected categories
- Includes sub-categories when filtering by parent

## Sorting

- Alphabetical by category name
- Click header to toggle direction
- Uncategorized items sort to end

## Budget Integration

- Categories link to budget allocations
- Over-budget categories may show warning indicators
- Category spending appears in analytics

## Tips

- Bulk select transactions to recategorize multiple at once
- Set category defaults on payees for automatic assignment
- Review uncategorized transactions regularly
- Category changes affect budget calculations immediately
