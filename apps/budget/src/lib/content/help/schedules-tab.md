# Schedules Tab

Set up and manage recurring transactions like bills, subscriptions, and regular transfers.

## Schedule Types

- **Bills** - Regular payments (rent, utilities)
- **Subscriptions** - Monthly services (Netflix, gym)
- **Income** - Regular deposits (salary, dividends)
- **Transfers** - Recurring account transfers

## Schedule Properties

| Field | Description |
|-------|-------------|
| Amount | Transaction value |
| Frequency | How often it repeats |
| Start Date | When to begin |
| End Date | When to stop (optional) |
| Payee | Who receives payment |
| Category | Budget category |

## Frequency Options

- Daily, Weekly, Bi-weekly
- Monthly (specific date or day)
- Quarterly, Yearly
- Custom intervals

## AI Recommendations

The **Recommendations** button shows:
- Detected recurring patterns
- Suggested new schedules
- Patterns from your transactions

## Managing Schedules

- **Edit** - Modify schedule details
- **Skip** - Skip next occurrence
- **Pause** - Temporarily stop
- **Delete** - Remove the schedule

## Tips

- Schedules create transactions automatically
- Review generated transactions before the due date
- Set reminders for variable bills
- Use for savings goals with regular transfers
