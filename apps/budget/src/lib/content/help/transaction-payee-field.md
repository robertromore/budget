# Payee Field

The payee is the merchant, vendor, or person involved in the transaction.

## Selection Methods

1. **Type to search** - Start typing to filter existing payees
2. **Create new** - Type a new name and press Enter to create
3. **Recent list** - Recent payees appear at the top

## Smart Features

- **Auto-complete** - Suggests matches as you type
- **Fuzzy matching** - Finds payees even with typos
- **Default categories** - Selecting a payee can auto-fill the category

## Payee Normalization

When AI intelligence is enabled:
- Similar names are grouped (e.g., "AMZN" and "Amazon")
- Merchant names are cleaned up automatically
- Duplicates are detected and suggested for merging

## Tips

- Leave blank if the payee doesn't matter for tracking
- Create consistent naming for better reports
- Use the Payees page to manage and merge duplicates
- Payee names are shared across all accounts
