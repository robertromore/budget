# Theme Colors

Customize the application's color scheme with different theme options.

## Opening Theme Selector

Click the **palette** icon in the header to open the theme picker.

## Available Themes

Choose from multiple color schemes:

- **Default** - Standard blue accent
- **Rose** - Pink/rose tones
- **Orange** - Warm orange accents
- **Green** - Nature-inspired green
- **Blue** - Classic blue theme
- **Violet** - Purple accents
- **Zinc** - Neutral gray tones

## Theme Components

Each theme affects:

- **Primary color** - Buttons, links, accents
- **Background** - Page and card backgrounds
- **Borders** - Dividers and outlines
- **Text** - Heading and body colors

## Preview

- Hover over themes to preview
- Click to apply
- Changes take effect immediately

## Light/Dark Mode

Themes work with both:

- Light mode
- Dark mode
- System preference

Use the **Theme Toggle** to switch between light/dark modes.

## Tips

- Choose high-contrast themes for accessibility
- Darker themes reduce eye strain at night
- Theme preference is saved per browser
