# Profile Name

A descriptive name for your import profile.

## Best Practices

Use names that identify:
- The bank or financial institution
- The specific account type
- Any distinguishing details

## Examples

- "Chase Checking"
- "Amex Gold Card"
- "Fidelity Brokerage"
- "PayPal Business"

## Tips

- Be specific enough to distinguish between accounts
- Keep names short but descriptive
- You can rename profiles anytime
