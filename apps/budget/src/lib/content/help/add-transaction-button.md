# Add Transaction Button

Click this button to manually enter a new transaction for the current account.

## When to Use

- Recording cash purchases not captured by bank imports
- Adding transactions before they appear in your bank
- Correcting or adjusting transaction amounts
- Creating scheduled future transactions

## Entry Methods

The dialog offers three ways to enter transactions:

1. **Manual** - Full control over all fields
2. **Transfer** - Move money between your accounts
3. **Guided** - Step-by-step wizard with suggestions

## Tips

- For recurring bills, consider setting up a schedule instead
- Use negative amounts for expenses, positive for income
- The payee field suggests matches as you type
- Categories can be auto-assigned based on payee defaults
