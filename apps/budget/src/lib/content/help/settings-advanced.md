# Advanced Settings

Advanced options for data management. Use with caution.

## Danger Zone

This section contains irreversible actions that can permanently delete your data.

### Delete All Data

Permanently removes all data from your workspace:

**What Gets Deleted:**
- All accounts and their transactions
- All budgets and allocations
- All categories (custom and default)
- All payees and their details
- All saved views and filters
- All AI recommendations
- All learned spending patterns
- All import profiles

**What's Preserved:**
- Your user account
- Your workspace settings
- Your appearance preferences

### How It Works

1. Click "Delete All Data"
2. Read the confirmation dialog carefully
3. Type the confirmation phrase exactly
4. Click confirm to proceed

### When to Use This

- Starting fresh with a clean workspace
- Removing test data before going live
- Preparing to import from a different system

## Important Warnings

- **This action cannot be undone**
- There is no way to recover deleted data
- Consider exporting your data first
- Make sure you really want to do this

## Tips

- Export important data before deleting
- Double-check you're in the correct workspace
- If unsure, contact support first
